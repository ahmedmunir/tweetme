# TweeTME:
TweeTME is a simpler version website of [Twitter](https://twitter.com/) website.  

## Website Features:
- Add Tweet.
- Edit Tweet.
- Delete Tweet.
- Retweet (You can retweet for a tweet, but you can't for a retweet).
- Like Tweet.
- Dislike Tweet.

## Features to be added in the future:
- Add photos with Tweet.
- Add reply to Tweet.
